Schnellstart Anleitung
So richtest du alles in 10 Minuten ein
SCHRITT 1
Brevo Account erstellen (kostenlos)
Geh auf brevo.com → "Sign up free" → Mit hertle.anjuli@praxisnovaai.com registrieren.

Danach: Settings → API Keys → "Create API Key" → Key kopieren und links eintragen.
SCHRITT 2
Brevo Kontaktliste erstellen
In Brevo: Contacts → Lists → "Create a list" → Name: "PraxisNova Outreach".

Die List-ID steht in der URL (z.B. /lists/3 → ID ist 3). Diese ID links eintragen.
SCHRITT 3
HubSpot API Key holen
In HubSpot: Settings → Account Setup → Integrations → Private Apps → "Create a private app".

Scope aktivieren: crm.objects.contacts.write · crm.objects.contacts.read
API Key kopieren und links eintragen.
SCHRITT 4
Konfiguration speichern und loslegen
Klicke auf "Konfiguration speichern".

Dann: Lead Finder → Suchbegriff eingeben → Leads finden → Mit einem Klick zu HubSpot CRM hinzufuegen UND E-Mail Sequence starten.
SCHRITT 5
Auf Vercel deployen (kostenlos)
Lade diese index.html in dein GitHub Repo praxisnova-fixed hoch (in einen neuen Ordner /leads-tool/).

Vercel deployed es automatisch. Deine eigene kostenlose Lead-Maschine ist live!
